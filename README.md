# PYVSM
- An implementation and visualizer of a virtual stack machine using Python3.
- The vsm is designed by Nagisa-Ishiura.

## Using Library
- pygame
  - `pip3 install pygame`

## Run
```
$ ./pyvsm.py <filename>
$ ./visualizer.py <filename>
```

## Visualizer key bind
- space: execute a step.
- q: quit
